Здесь лежат комманды для BotFather в том формате, который от требует (для упрощения редактирования)

------------------------------------------------------------------------------------------------------------
Moderation
get_all_users_csv - Скачать csv файл с данными о пользователях
get_all_requests_csv - Скачать csv файл с данными о задачах
assign_moderator - Назначить пользователя модератором (после команды укажите id пользователя)
assign_admin - Назначить пользователя админом (после команды укажите id пользователя)
withdraw_moderator - Лишить пользователя статуса модератора (после команды укажите id пользователя)
withdraw_admin - Лишить пользователя статуса админа (после команды укажите id пользователя)
all_tags - Получить список кодов всех тэгов и их статусов
tag_info - Подробная информация о тэге (после команды укажите код тэга)
change_tag_condition - На противоположный: отображается/не отображается (после команды укажите код тэга)
new_tag - ФОРМАТ АРГУМЕНТОВ: код 'название на русском' 'название на английском'
delete_tag - после команды в том же сообщении укажите код тэга
